## Übersicht

Das **Ausgehend-Dashboard** (Outbound) zeigt alle Ihre Verkaufsartikel und deren Compliance-Status. Hier können Sie optional eingehende Artikel (Input) mit ausgehenden Artikeln (Output) verknüpfen, um DDS-Nachweise für nachgelagerte Marktteilnehmer (Downstream Operators) bereitzustellen.

<Frame caption="Ausgehend-Dashboard mit Verkaufsartikeln">
  <img src="/images/helpcenter/optional/outbound-dashboard.png" />
</Frame>

<Info>
Die Outbound-Artikelverknüpfung ist **optional**. Sie wird benötigt, wenn Sie Waren weiterverarbeiten oder weiterverkaufen und dabei die EUDR-Compliance-Nachweise der Eingangsartikel auf Ihre Ausgangsartikel übertragen möchten.
Außerdem ist es zurzeit noch möglich eine DDS für Inländische Produktionen enzureichen.
</Info>

## Wann ist diese Verknüpfung sinnvoll?

**Beispiele:**
- Kakaobohnen (Einkaufsartikel) → Schokolade (Verkaufsartikel)
- Mehrere Rohstoffe werden zu einem Fertigprodukt kombiniert
- Ein Artikel wird umverpackt und unter neuer Artikelnummer verkauft

## Eingangsartikel verknüpfen (BOM-Linking)

<Steps>
  <Step title="Verkaufsartikel auswählen">
    Navigieren Sie im **Ausgehend-Dashboard** zum Verkaufsartikel, dem Sie Eingangsartikel zuordnen möchten.
  </Step>
  <Step title="Eingangsartikel verknüpfen">
    Klicken Sie auf **"Eingänge verknüpfen"**. Es öffnet sich ein Dialog, in dem Sie aus Ihren Einkaufsartikeln auswählen können.
  </Step>
  <Step title="Einkaufsartikel auswählen">
    Suchen Sie nach Name oder Artikelnummer und wählen Sie die Artikel aus, die als Input für diesen Verkaufsartikel verwendet werden.
  </Step>
  <Step title="Stückliste speichern">
    Klicken Sie auf **"Stückliste speichern"**. Die Compliance-Nachweise der Eingangsartikel werden automatisch auf den Verkaufsartikel übertragen.
  </Step>
</Steps>

## EU-Produktion (Domestic Production)

Falls Sie Waren in der EU selbst produzieren, können Sie direkt im Ausgehend-Dashboard die **EU-Produktionsinformationen** hinterlegen:

<Steps>
  <Step title="Produktionsland angeben">
    Wählen Sie das Produktionsland und speichern Sie.
  </Step>
  <Step title="Produktionsfläche hinzufügen">
    Zeichnen Sie die Fläche auf der Karte ein oder laden Sie eine GeoJSON-Datei hoch.
  </Step>
  <Step title="DDS erstellen">
    Geben Sie die Menge (kg) an, akzeptieren Sie die Erklärung und reichen Sie die DDS für inländische Produktion ein.
  </Step>
</Steps>

<Frame caption="Indländische Produktion">
  <img src="/images/helpcenter/optional/outbound-prod.png" />
</Frame>

<Note>
Die Compliance-Nachweise der Eingangsartikel müssen vollständig sein, bevor die Verknüpfung mit Verkaufsartikeln sinnvoll ist.
Externe DDS von Lieferanten können über diese Verknüpfung auch autoamtisch an verkaufte Artikel übertragen werden.
</Note>
