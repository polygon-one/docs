## Übersicht

Die **Kartenansicht** bietet eine globale Übersicht über alle erfassten Produktionsflächen (Plots) in Polygon One.

## Funktionen

- **Globale Übersicht** — Alle Produktionsstandorte Ihrer Lieferanten auf einer interaktiven Karte
- **Flächen-Sidebar** — Liste aller Flächen mit Suchfunktion
- **Filter** — Nach Status, Rohstoff oder Lieferant filtern
- **Flächendetails** — Klicken Sie auf eine Fläche für Details:
  - Flächengröße (in Hektar)
  - Koordinaten
  - Zugehörige Artikel
  - Compliance-Status (Kein Entwaldungsrisiko / Entwaldungsrisiko erkannt / Analyse ausstehend)
  - Satellitenanalyse-Ergebnisse (ESA, JRC, GFT, TMF, PALSAR)
  - Risikokennzeichen

## Aktionen auf der Karte

- **Fläche zeichnen** — Neue Produktionsflächen direkt auf der Karte einzeichnen
- **GeoJSON hochladen** — Geodaten als Datei importieren
- **Fläche löschen** — Bestehende Flächen entfernen
- **Grenzen bearbeiten** — Flächengrenzen durch Ziehen der Eckpunkte anpassen
- **Adresssuche** — Nach einer Adresse suchen und auf der Karte navigieren

<Tip>
Nutzen Sie die Kartenansicht für einen schnellen visuellen Überblick über die Verteilung Ihrer Lieferkette und die Identifizierung von Risikogebieten.
</Tip>
