Bevor die EUDR-Analyse starten kann, müssen Ihre Stammdaten in Polygon One hinterlegt werden. Dazu gehören Ihre **Lieferanten** (mit Kontaktdaten) und Ihre **Artikel** (mit HS-Codes). Beide können per CSV-Import, manuell oder über die REST API angelegt werden. Sobald Artikel mit HS-Codes vorhanden sind, prüft das System automatisch die EUDR-Relevanz.
