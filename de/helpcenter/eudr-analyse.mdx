## Übersicht

Nach dem Import Ihrer Artikel führt Polygon One automatisch eine **EUDR-Relevanzanalyse** durch. Das System prüft anhand der HS-Codes, welche Ihrer Artikel unter die EU-Entwaldungsverordnung fallen.

## Automatische Analyse

Die Analyse erfolgt automatisch im Hintergrund, sobald Artikel mit einem HS-Code in der Plattform angelegt werden. Sie müssen hierfür nichts manuell anstoßen.

### Einteilung der Artikel

Nach der Analyse werden Ihre Artikel gekennzeichnet:

| Kennzeichnung | Beschreibung |
|---------------|-------------|
| **EUDR-relevant** | Der Artikel enthält Rohstoffe, die unter die EUDR fallen. Für diesen Artikel müssen Nachweise gesammelt und eine DDS erstellt werden. |
| **Nicht EUDR-relevant** | Der Artikel fällt nicht unter die EUDR. Es sind keine weiteren Maßnahmen erforderlich. |

Die EUDR-Relevanz wird in der Artikelübersicht als Spalte angezeigt und kann als Filter verwendet werden.

## Compliance-Dashboard

Sobald die Analyse abgeschlossen ist, zeigt das **Compliance-Dashboard** (Eingehend) eine vollständige Übersicht:

<Frame caption="Compliance-Dashboard mit Übersichtskennzahlen">
  <img src="/images/helpcenter/eudr-analyse/eudr-analyse-dashboard.png" />
</Frame>

- **EUDR-relevante Lieferanten** — Anzahl der betroffenen Lieferanten
- **Analysierte Fläche** — Gesamtfläche in Hektar
- **Konforme Artikel** — Anzahl der konformen EUDR-relevanten Artikel
- **Ausstehende Aktionen** — Artikel mit fehlenden Geo-Daten, fehlender rechtlicher Analyse oder fehlendem Bericht

### Compliance-Status pro Artikel

| Status | Beschreibung |
|--------|-------------|
| **Konform** | Alle Anforderungen erfüllt — bereit für DDS |
| **Nicht konform** | Entwaldungsrisiko oder rechtliche Mängel erkannt |
| **Prüfung ausstehend** | Daten wurden eingereicht, Analyse läuft |
| **Unbekannt** | Fehlende Daten oder Dokumentation |
